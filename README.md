# vanilla-todo-crud
Simple yet powerful Todo app showcasing full CRUD operations. Includes dynamic filtering, inline editing, and a modern UI. No frameworks, just pure HTML, CSS, and JavaScript.

## Features
- Create, Read, Update, Delete (CRUD) tasks
- Filter tasks by status (All, Active, Completed)
- Inline editing of task titles
- Responsive design for mobile and desktop
- Clean and modern UI with CSS variables and animations
## Usage
1. Clone the repository:
   ```bash
   git clone https://github.com/rahafebx/vanilla-todo-crud.git
    ```
2. Open `index.html` in your web browser.
3. Start adding, editing, and managing your tasks!
